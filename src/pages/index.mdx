<summary>[Dijkstra's Algorithm](https://en.wikipedia.org/wiki/Dijkstra%27s_algorithm)</summary>

```rs
fn dijkstra(x: usize, y:usize, graph: Vec<Vec<(usize,usize)>>) -> Option<usize>{
    // set all distance to MAX
    let mut d = vec![!0; graph.len()];
    // set dist(x, x) to 0
    d[x] = 0;
    // create priority queue
    let mut pq = BinaryHeap::from([Reverse((0, x))]);
    // check the node with the lowest weight first (min-heap)
    while let Some(Reverse((w, wu))) = pq.pop() {
        // if reach destination (shortest)
        if u == y { return Some(wu); }
        // if current path is not the shortest
        if wu > d[u] { continue; }
        //          new weight for x -> u -> v
        graph[u].iter().map(|&(v, wv)| (v, wu + wv)).for_each(|(v, wv)| {
            // if new weight is shorter
            if wv < d[v] {
                pq.push(Reverse((wv, v)));
                d[v] = wv;
            }
        });
    }
    None
}
```

<summary>Prove that $L=\{10^k1:k\textsf{ is prime}\}$ is **not** a [regular language](https://en.wikipedia.org/wiki/Regular_language)</summary>

By the way of contradiction, suppose $L$ is regular\
$\therefore\exists n\ge 1$ as the "[pumping length](https://en.wikipedia.org/wiki/Pumping_lemma_for_regular_languages#Formal_statement)"

let $x=10^p1$ where $p>n$ is a prime\
$\therefore x\in L$ by definition, and $|x|=p+2>p>n$\
$\therefore\exists u,v,w\in\Sigma^\ast$ such that
1. $x=uvw$
2. $v\ne\varepsilon$
3. $|uv|\le n$
4. $\forall k\in\mathbb N\enspace uv^kw\in L$

however, case $u=\varepsilon$
$$
\begin{align*}
  &\because u=\varepsilon\\
  &\therefore\exists i\in[0,n)\textsf{ st }v=10^i\enspace w=0^{p-i}1\\
  &\because uv^0w=w=0^{p-1}1\not\in L\\
  &\therefore\textsf{contradicts 4.}\\
\end{align*}
$$

case $u\ne\varepsilon$
$$
\begin{align*}
  &\because u\ne\varepsilon\\
  &\therefore\exist i,j\in(0,n)\textsf{ st }i<j\quad u=10^{i-j}\quad v=0^j\quad w=0^{p-i}1\\
  &\therefore uv^{p+1}w=10^{i-j}0^{(p+1)j}0^{p-i}1=10^{p(j+1)}1\\
  &\because p\textsf{ is prime}\quad j\in\mathbb N\implies j+1>0\\
  &\therefore p(j+1)\textsf{ is not prime}\\
  &\therefore uv^{p+1}w=10^{p(j+1)}1\not\in L\\
  &\therefore\textsf{contradicts 4.}
\end{align*}
$$

$\therefore L$ is not regular
</details>
